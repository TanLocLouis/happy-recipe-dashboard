# Overview
- This is dashboard for happy-recipe project in Introduction to Software Engineering.

# Tech stack
- React + TS.

# Installation
1. Clone this project.
```
git clone https://github.com/TanLocLouis/happy-recipe-dashboard
```
2. Config `.env` file.
2. Install node_modules and dependencies.
```
npm install
```
3. Run.
```
npm start
```
